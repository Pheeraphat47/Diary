เป็นอีกหนึ่งโครงการที่ดีมากๆ ผมชอบ Concept ของโครงการตรงที่เค้าเปิดโอกาสให้รุ่นพี่ ได้มาสอนน้องๆ แทนที่จะเอาอาจารย์มาสอน มันดีตรงที่พอเป็นรุ่นพี่แล้ว สำหรับผมมันทำให้รู้สึกไม่เกร็ง ดูแล้วรุ่นพี่หลายๆคนค่อนข้างที่จะเอนเตอร์เทนน้องๆเก่งมากก คือรู้เลยว่าพยายามไม่ให้มีเดดแอร์เกิดขึ้น ซึ่งมันก็ดีนะคับ แต่ผมอยากจะบอกว่าพี่ๆพักกันบ้างก็ได้ระคับ ผมเหนื่อยแทน 555 และก็พี่ๆทุกคนคอยเป็นห่วงน้องๆทุกคนอยู่ตลอดเลยฮะ อย่างวันแรกก็มีเพื่อนผมที่ชื่อ เฟิรส์ อยู่ดีๆเค้าก็หูดับไป พี่ๆก็คอย takecare ดีมากๆ โมเมนต์นะตอนนั้นคือประทับใจมาก และก็ระหว่างที่พี่ๆสอนอยู่พอมีเพื่อนๆในกลุ่มไมเข้าใจ พี่ๆก็เข้ามาแทบจะทันทีเลย ตรงนี้ก็เป็นอีกจุดนึงที่ผมประทับใจ และโครงการนี้ยังทำให้ผม ได้รู้จักกับเพื่อนใหม่ๆอีกหลายๆคนที่ผมยังไม่เคยคุยด้วย รวมไปถึงพี่ๆทุกคนที่อยู่ในโครงการอีกด้วย หลังจบโครงการผมก็ได้คอนแทคของหลายๆคนมาเพียบเลย ก็ต้องขอบคุณโครงการนี้อีกเช่นเคย และเนื้อหาที่สอนทั้งหมดในโครงการก็เป็นเนื้อหาที่ผมจะได้เจอในการเรียนจริงอีกด้วยไม่ว่าจะเป็น การเขียนภาษา Java , Html , Css , Sql แล้วก็ Git เป็นต้น ซึ่งพี่ๆทุกคนก็เก่งมาก สอนดีมาก ผมเข้าใจในสิ่งที่พวกพี่สอนหมดเลยนะ แล้วก็ยังได้คำแนะนำดีๆในการเรียนจากรุ่นพี่อีก คือรุ่นพี่เล่าแบบหมดเปลือกเลย555 ก็อยากจะขอบคุณพี่ๆทุกคนมากเลยค้าบ นี่เป็นหนึ่งในค่ายที่ดีที่สุดในชีวิตผมเลย
