My Diary

IT#28 StarterPack (1/8/2022 - 6/8/2022)

เป็นโครงการปรับพื้นฐานของสาขาเทคโนโลยีสารสนเทศ ที่เปิดโอกาสให้น้องๆปี 1 ได้มาเรียนปรับพื้นฐานกับพวกพี่ๆปี 2 ปี 3 โดยที่ไม่มีอาจารย์มากเกี่ยวข้อง เป็นโครงการที่เปิดโอกาสให้รุ่นน้องที่พึ่งเข้ามาใหม่ได้รู้จักกับเพื่อนใหม่ๆในสาขารวมถึงพี่ๆทุกคนมากยิ่งขึ้น และเนื้อหาที่สอนทั้งหมดในโครงการ ก็เป็นเนื้อหาที่ผมจะได้เจอในการเรียนจริงอีกด้วย ไม่ว่าจะเป็น การเขียนภาษา Java Html Css แล้วก็ Git เป็นต้น

CAI CLUB HACKATHON (20/8/2022 - 21/8/2022)

เป็นโครงการที่เปิดโอกาสให้เราได้นำใอเดียหรือนวัตกรรมที่เกี่ยวกับ AI ของเรา มาพัฒนาเป็น Prototype ภายในระยะเวลา 2 วัน 1 คืน ร่วมกับเพื่อนๆในโครงการ โดยเราก็จะแยกกันเป็นกลุ่มๆตามเรื่องที่เราสนใจ และภายในกิจกรรมเองก็มีอีกหลาย session อย่างเช่น Workshop กับ Mentoring และตอนท้ายเราก็จะได้นำใอเดียของเานำไป Pitching ให้กรรรมการที่มาจากบริษัทภาคเอกชนต่างๆฟังอีกด้วย

FRESHY BAND #9 (20/10/2022)

เป็นการประกวดวงดนตรีสำหรับน้องๆนักศึกษาชั้นปีที่ 1 รหัส 65 ซึ่งจัดโดยชมรมบันเทิงและดนตรสากลของมหาวิทยาลัย เทคโนโลยีพระจอมเกล้าธนบุรี โดยผมเองได้เข้าร่วมประกวดในงานนี้ในฐานะสมาชิกของวง Gumairuu โดยผมเล่นเป็นมือเบสของวง

ENVIRONHACK CLIMATHON BANGKOK (28/10/2022 - 30/10/2022)

เป็นโครงการที่เปิดโอกาสให้ผู้ที่มีอายุตั้งแต่ 17 - 35 ปี ทุกคน ได้มีโอกาสออกใอเดียร่วมกันแก้ไขปัญหาสิ่งแวดล้อม โดยเราต้องทำงานร่วมกับผู้สมัครคนอื่นๆภายในงานนี้ และเราต้องหาทางแก้ไขปัญหา ที่ทีมงานเค้ามอบให้ ภายในระยะเวลา 3 วัน 2 คืน ผ่านกระบวนการ Hackathon ผลิตต้นแบบออกมาเพื่นนำเสนอให้คณะกรรมการ ซึ่งใอเดียที่ได้รับรางวัลก็จะมีโอกาสได้นำไปใ้ช้จริงอีกด้วย

SIT HELLOWORLD (29/10/2022 - 26/11/2022)

เป็นโครงการของคณะเทคโนโลยีสารสนเทศ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี โดยจะเปิดโอกาสให้พี่ๆปี 2 ปี 3 มาสอนน้องๆปี 1 โดยจะแยกออกเป็น 4 ฝ่ายหลักๆ ได้แก่ ฝ่าย Frontend ฝ่าย Web Design ฝ่าย Ingrastructure และก็ฝ่าย Game Dev ซึ่งน้องๆปี 1 ทุกคนก็สามารถเลือกสมัครเข้าโครงการได้ตามความสนใจ โดยเมื่อสมัครไปแล้ว ก็จะมีพี่ๆมาคัดเลือกใบสมัครของน้องอีกที เพื่อคัดคนเข้าโครงการ และถ้าน้องๆปี 1 ผ่านการคัดเลือก ก็จะได้เข้าโครงการไปเรียนกับพวกพี่ๆ รวมถึงได้ทำโปรเจคร่วมกันกับเพื่อนๆในโครงการอีกด้วย
